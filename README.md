<h1>🎨 Paint App</h1>

<p>A simple and interactive <strong>Paint Application</strong> built using <strong>HTML, CSS, and JavaScript</strong>. This app allows users to draw freehand with a brush, create shapes (circle, rectangle, triangle), select colors and brush sizes, erase unwanted parts, clear the canvas, and save their artwork as an image.</p>

<h2>🚀 Features</h2>

<ul>
  <li>✅ <strong>Brush Tool</strong> – Draw freely on the canvas</li>
  <li>✅ <strong>Shape Drawing</strong> – Draw circles, triangles, and rectangles</li>
  <li>✅ <strong>Fill Color Option</strong> – Toggle between filled and outlined shapes</li>
  <li>✅ <strong>Custom Colors & Width</strong> – Select any color and adjust brush/shape width</li>
  <li>✅ <strong>Eraser</strong> – Remove unwanted strokes</li>
  <li>✅ <strong>Clear Canvas Button</strong> – Reset the canvas with a single click</li>
  <li>✅ <strong>Save as Image</strong> – Export your drawing as an image file</li>
  <li>✅ <strong>Responsive UI</strong> – Works smoothly across different screen sizes</li>
</ul>
